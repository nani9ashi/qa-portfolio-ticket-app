## 1. 目的
本ドキュメントは、本ポートフォリオでのテスト実行に用いる環境（OS/ブラウザ/実行形態など）を明確にし、結果の再現性を高めることを目的とする。

## 2. テスト実行環境（想定）
- 実行形態：ローカル実行（例：Docker）またはローカル起動
- OS：Windows 10/11
- ブラウザ：
  - Google Chrome（主）
  - Microsoft Edge（副）
- 画面サイズ：PC表示（例：1366×768以上）
- ネットワーク：通常回線（特別な制御は行わない）

## 3. テストデータ（アカウント）
本リポジトリ内のテストデータは架空の値を用いる。
- 依頼者：req01
- 担当者：agent01
- 管理者：admin01

※パスワードやトークン等の機密情報はリポジトリに含めない。

## 4. 証跡（Evidence）の扱い
- スクリーンショット：`../evidence/screenshots/`
- 動画：`../evidence/videos/`
- ログ：`../evidence/logs/`

容量が大きい証跡は、必要に応じて圧縮または別手段で共有し、実行記録の「証跡」列から参照できるようにする。

## 5. 既知の制約
- 本環境はポートフォリオ用途のため、負荷試験・本番相当構成の再現は対象外とする。
